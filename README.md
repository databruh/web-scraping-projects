# Web Scraping & Data Extraction Projects

Python-based web scraping and data processing projects built to demonstrate practical skills in **data extraction, cleaning, validation, and structured output** — directly relevant for AI Pilot / Vibe Code Specialist (Tendem Project) roles at Mindrift.

### Projects

**1. E-commerce Product Data Extractor**
- Scrapes product information from multi-page e-commerce sites using `BeautifulSoup` + `Requests`.
- Handles data cleaning, normalization, validation, and exports to CSV + JSON.

**2. Dynamic Job Listings Scraper**
- Uses `Selenium` + `BeautifulSoup` to handle JavaScript-rendered pages, infinite scroll, and dynamic content.
- Extracts hierarchical data, performs consistency checks, and saves structured datasets.

### Technologies Used
- Python 3
- BeautifulSoup4, Requests
- Selenium (for dynamic content)
- Pandas (data cleaning & validation)
- JSON, CSV export
